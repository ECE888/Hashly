# Hashly
Hasly est un projet universitaire qui détecte automatiquement l'agorithme de hashing d'un hash donné, qui propose une méthode adaptée de craking et génère une commande prête à l'emploi pour des outils comme Hashcat ou John The ripper.

Hashly is a college cybersecurity project that automatically detects the hashing algorithm of a given hash, suggests the best cracking method, and generates ready-to-use commands for tools like Hashcat or John the Ripper.
